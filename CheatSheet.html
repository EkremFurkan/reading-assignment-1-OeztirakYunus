World Wide Web General

1. What are the most essential technologies that made the World Wide Web such a success?
A.:The most essential technologies are URL (Uniform Resource Locator), HTML (HyperText Markup Language) and HTTP (HyperText Transfer Protocol).

2.How are resources (html pages, images, etc) addressed in the web?
A.: It is addressed like http://www.example.com/index.html, which indicates a protocol (http), a hostname (www.example.com), and a file name (index.html).

3. What are the general components of a url?
A.: The general components of a url are a scheme, a host, a path and a query string.
